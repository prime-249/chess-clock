# Chess Clock

An Arduino-based, fully functional chess clock.

## Features

- Two independent but synchronized timers (White / Black)
- Button-controlled turn switching
- Continuous and accurate time tracking
- Extensive selection of time controls with increment
- Pause feature

## Hardware Requirements

- Arduino UNO board
- Push buttons
- Four 7-segment displays
- Basic electronic components (resistors, wires, breadboard)

## Software Overview

- Written in Arduino C++
- Time management based on non-blocking logic (`millis()`)
- Clear separation between clock logic and hardware interaction
- Object-Oriented Programming
